Sanky Diagram¶

  • Descripció: els diagrames Sanky mostren la relació entre diferents variables o categories, i la quantitat de flux que passa entre elles. Són gràfics de xarxes que mostren la direcció del flux i la seva amplada representa la quantitat de flux que hi passa. Els diagrames Sanky són especialment útils per representar dades sobre la relació entre diferents categories i el flux que hi passa, i poden ser utilitzats per a qualsevol variable o categoria.
  • Exemples d'aplicació: tràfic web (visites a les pàgines web), contaminacions de països, etc.
  • Tipus de dades: aquesta tècnica és adequada per a dades quantitatives i també pot ser utilitzada per a dades categòriques, ja que mostra la relació entre categories.
  • Limitacions: els diagrames Sanky són adequats per a conjunts de dades mitjans o grans, ja que si la quantitat de dades és petita, el diagrama pot resultar poc informatiu o difícil de llegir.

# seleccionem i filtrem les variables que ens interessa graficar
df = pd.read_csv('Methane_final.csv', encoding = "UTF-8")
df = df[['region', 'type', 'emissions', 'baseYear']]
df = df.groupby(['region', 'type', 'baseYear'], as_index=False).sum()
categorias = df['region'].unique().tolist() + df['type'].unique().tolist() + df['baseYear'].unique().tolist()

# crear conexions entre les categoríes anteriors seleccionades
conexiones = []
for i in df.index:
    conexion = {
        'source': categorias.index(df['region'][i]),
        'target': categorias.index(df['type'][i]),
        'value': df['emissions'][i],
        'baseYear': df['baseYear'][i]
    }
    conexiones.append(conexion)

# crearem un DataFrame per les categorías i l'altre per a les conexions
df_cat = pd.DataFrame(categorias)
df_cat.columns = ['categoria']
df_con = pd.DataFrame(conexiones)

# Montem la figura del diagrama
fig = go.Figure(data=[go.Sankey(
    node = dict(
        pad = 15,
        thickness = 20,
        line = dict(color = "black", width = 0.5),
        label = df_cat['categoria'].tolist(),
    ),
    link = dict( # customitzem els labels dels fluxos
        source = df_con['source'].tolist(),
        target = df_con['target'].tolist(),
        value = df_con['value'].tolist(),
        label = df_con['value'].tolist(),
        customdata = df_con['baseYear'].tolist(),
        hovertemplate = 'De: %{source.label}<br>A: %{target.label}<br>Value: %{value}<br>Any: %{customdata}'
    ))])

# la distribució de les emissions de CO2 per sector i regió
fig.update_layout(title="Distribució de les emissions de Metà (CH) per sector i regió, classificat per any (2019-2022) d'emissió",
                  font=dict(size=12, color='black'), 
                  height=800, width=1000)
# Subtitle de la font
fig.add_annotation(
    xref="paper",
    yref="paper",
    x=0.005,
    y=-0.05,
    text="Font: Kaggle - Dataset - Global Methane Emissions",
    showarrow=False,
    font=dict(size=12),
)
fig.show(renderer='notebook')

Waterfall Chart¶

  • Descripció: els gràfics de cascada mostren la distribució dels valors d'una variable a través del temps o de diferents fases d'un procés. El gràfic comença amb un valor inicial, que es representa per una barra, i després es mostren les diferents fases o factors que van afectant aquest valor, tant positius com negatius. Els valors s'apilen uns damunt dels altres, formant una estructura similar a la de les cascades. Aquest tipus de gràfic és ideal per a la comparació entre diferents fases o períodes.
  • Exemples d'aplicació: finances (ingressos i despeses en diferents fases), rendiment d'una empresa (anàlisi de beneficis i pèrdues), etc.
  • Tipus de dades: és adequada per a dades quantitatives i també pot ser utilitzada per a dades categòriques, ja que mostra la distribució dels valors.
  • Limitacions: els gràfics de cascada són adequats per a conjunts de dades mitjans o grans, ja que si la quantitat de dades és petita, el gràfic pot resultar poc informatiu o difícil de llegir.

Rain cloud Plots¶

  • Descripció: els gràfics de núvol de pluja mostren la distribució de les dades a través de diferents grups o categories. Aquesta tècnica combina les funcionalitats dels boxplots, les densities plots i les dot plots. Els núvols de pluja mostren les dades en forma de núvol, on la seva densitat representa la freqüència relativa de les dades. A més a més, també mostren les dades en forma de boxplots, que mostren la mediana, quartils i els valors mínim i màxim, i en forma de dot plots, que mostren cada valor individual. Això fa que els núvols de pluja siguin una eina útil per mostrar la distribució de les dades d'una forma completa i clara.
  • Exemples d'aplicació: al camp de la psicologia (anàlisi de dades de conductes o emocions), medicina (anàlisi de dades clíniques), etc.
  • Tipus de dades: és adequada per a dades quantitatives, ja que mostra la distribució de les dades. També pot ser utilitzada per a dades categòriques, però en aquest cas, les dades s'han de transformar en valors numèrics per poder ser representades.
  • Limitacions: els núvols de pluja són adequats per a conjunts de dades mitjans o grans, ja que si la quantitat de dades és petita, el gràfic pot resultar poc informatiu o difícil de llegir. També pot ser difícil interpretar els núvols de pluja si les distribucions de les dades són molt diferents entre els grups o categories.

dades <- read.csv("2022_IRIS_Peticions_Ciutadanes_OpenData.csv", encoding = "UTF-8")
dades <- dades %>% filter(TIPUS  %in% c("INCIDENCIA") & AREA %in% c("Sanitat i salut pública", "Transports públics"))

# Crear variables de temps de resolució per cada incidència
dades$DIA_DATA_ALTA <- as.Date(dades$DIA_DATA_ALTA, format = "%d/%m/%Y")
dades$DIA_DATA_TANCAMENT <- as.Date(dades$DIA_DATA_TANCAMENT, format = "%d/%m/%Y")
dades$TEMPS_RESOLUCIO <- abs(difftime(dades$DIA_DATA_TANCAMENT, dades$DIA_DATA_ALTA, units = "days"))


# generem el plot Rainy Cloud
ggplot(dades, aes(y = TEMPS_RESOLUCIO, x = AREA, fill = AREA)) + 
    geom_flat_violin(position = position_nudge(x = .2, y = 0)) +
    coord_flip() + 
    geom_jitter(alpha = 0.5,width = 0.15) +
    theme(legend.position = "none") +
    geom_boxplot( width = .25,  outlier.shape = NA ) +
    labs(title = "Temps de resolució d'incidències per àrea",
         y = "Temps de resolució (dies)",
         x = "Àrea",
         subtitle = "Font: 2022_IRIS_Peticions_Ciutadanes_OpenData.csv") +
    scale_fill_manual(values = c("Sanitat i salut pública" = "#0072B2", "Transports públics" = "#009E73"))
